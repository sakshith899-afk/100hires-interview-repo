What I Did

This is my first time setting up a developer environment.
I am not from a coding background so some of these steps
were new to me. I have written everything I did and the
problems I ran into.


Tools I Installed

1. Cursor IDE
2. Claude Code extension (by Anthropic)
3. Codex extension (by OpenAI)


Steps I Followed

1. Installing Cursor

I went to cursor.com and downloaded it. The installation
was simple. During sign up I connected it with my GitHub
account which helped me in the later steps.

I did not know what an IDE was so I looked it up. It stands
for Integrated Development Environment. Basically it is a
place where developers write code and it has multiple tools
built into one app.

2. Adding Extensions

After opening Cursor I explored the menus and found
Extensions under the View tab. I searched for Claude Code
and installed it. Then I searched for Codex and installed
that too.

3. Creating a GitHub Repository

I already had a GitHub account so this part was easy.
I created a new public repository.

4. Opening the Repository in Cursor

This is where I got stuck. I did not know how to connect
GitHub to Cursor. I researched and found out I need to
use a clone command in the terminal with my repository link.

But when I tried it I got an error. Git was not installed
on my computer. So I downloaded it from git-scm.com and
installed it. The error still showed up after that.
I tried a few things and eventually just restarted Cursor.
After the restart it worked fine.

5. Writing this README

After cloning I opened the folder in Cursor and found the
README file and started writing everything I did here.
